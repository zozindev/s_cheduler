# 슼케줄러 (S-cheduler) - v2.0

**슼케줄러 (S-cheduler)**는 윈도우 기본 작업 스케줄러의 복잡함을 벗어나, 파이썬을 이용해 직관적이고 강력한 자동화 환경을 제공하는 멀티 태스크 스케줄링 도구입니다. 시스템 절전 모드 해제(Wake-up) 및 방지(Stay-Awake) 기능을 포함하여, 지정된 시간에 배치 파일이나 실행 파일을 자동으로 구동하고 결과를 이메일로 리포팅합니다.

---

## 🛠 기술 스택 (Tech Stack)

*   **Language**: Python 3.x
*   **GUI Framework**: `tkinter` (표준 라이브러리)
*   **System API**: Win32 API (`ctypes`를 통한 kernel32.dll, shell32.dll 연동)
*   **Process Management**: `subprocess`
*   **Communication**: `smtplib` (Gmail SMTP), `email.mime`
*   **Configuration**: JSON, `python-dotenv`

---

## ✨ 핵심 기능 (Key Features)

1.  **시스템 웨이크업 (Wake-up Timer)**:
    *   PC가 절전 모드 상태일 때, 하드웨어 타이머를 설정하여 작업 실행 직전에 시스템을 자동으로 깨웁니다. (Win32 API `SetWaitableTimer` 활용)
2.  **절전 모드 진입 방지 (Stay-Awake)**:
    *   프로그램 실행 중 PC가 자동으로 잠들지 않도록 제어하는 토글 기능을 제공합니다. (관리자 권한 불필요)
3.  **멀티 스케줄링 관리**:
    *   JSON 기반의 데이터 구조를 통해 여러 개의 작업을 독립적으로 관리합니다.
    *   작업별 실행 시각, 파일 경로, 절전 해제 여부, 개별 수신 이메일 설정을 지원합니다.
4.  **범용 실행 파일 지원**:
    *   `.bat`, `.cmd` 배치 파일뿐만 아니라 `.exe`, `.py` 등 실행 가능한 모든 파일을 지원합니다.
5.  **이메일 리포팅 시스템**:
    *   작업 완료 후 성공/실패 여부, 실행 로그(stdout/stderr), 반환 코드를 지정된 다수의 수신자에게 즉시 발송합니다.
6.  **직관적인 GUI**:
    *   스케줄 목록 조회, 추가, 수정, 삭제 기능을 시각적으로 제공하여 누구나 쉽게 사용 가능합니다.

## 🚀 시작하기 (Installation & Setup)

### 1. 환경 설정
프로그램 실행을 위해 필요한 패키지를 설치합니다.
```bash
pip install -r requirements.txt
```

### 2. 보안 설정 (.env)
이메일 알림 기능을 위해 프로젝트 루트에 `.env` 파일을 생성하고 Gmail 계정 정보를 입력합니다. (Gmail은 '앱 비밀번호' 사용이 필수입니다.)
```env
GMAIL_USER=your_email@gmail.com
GMAIL_APP_PASSWORD=your_google_app_password
```

### 3. 윈도우 전원 설정 (중요)
절전 모드 해제 기능이 정상 작동하려면 다음 설정이 필요합니다.
*   **제어판 > 전원 옵션 > 설정 변경 > 고급 전원 관리 옵션 설정 변경**
*   **절전 > 절전 모드 해제 타이머 허용 > '사용'**으로 설정

## 💡 사용 방법 (Usage Guide)

### 1. 프로그램 실행
**시스템 웨이크업 기능을 사용하려면 반드시 "관리자 권한"으로 명령 프롬프트나 터미널을 실행한 후 프로그램을 구동하세요.**
```bash
python src/main.py
```

### 2. 스케줄 등록 및 수정
1.  메인 화면에서 **[스케줄 추가]** 버튼을 클릭합니다.
2.  **작업 이름**: 고유한 이름을 입력합니다.
3.  **실행 시각**: `HH:MM` 형식으로 입력합니다 (예: 09:30).
4.  **파일 경로**: **[찾기]** 버튼을 눌러 실행할 파일을 선택합니다.
5.  **절전 해제 기능**: 해당 작업 실행 시 PC를 깨울지 여부를 체크합니다.
6.  **수신 이메일**: 결과를 받을 이메일 주소를 쉼표(,)로 구분하여 입력합니다.
7.  **[저장]**을 누르면 스케줄링이 즉시 시작됩니다.

### 3. 시스템 전원 제어
*   메인 화면 하단의 **"프로그램 실행 중 시스템이 자동으로 절전 모드에 진입하지 않도록 방지"** 체크박스를 켜면, 프로그램이 켜져 있는 동안 PC가 잠들지 않습니다.

## 📂 프로젝트 구조 (Project Structure)

*   `src/main.py`: 프로그램 진입점 및 백그라운드 엔진
*   `src/gui_manager.py`: 사용자 인터페이스 관리
*   `src/core/`: 전원 제어(`power_manager.py`) 및 작업 실행(`executor.py`) 핵심 로직
*   `src/utils/`: 설정 관리(`config_manager.py`) 및 메일 전송(`mail_sender.py`)
*   `data/`: `s_cheduler_config.json` 스케줄 저장소
*   `logs/`: 실행 로그 파일 저장

---

## ⚠️ 주의사항
*   이 프로그램은 **Windows 10/11** 환경에 최적화되어 있습니다.
*   Gmail 보안 정책에 따라 반드시 2단계 인증 후 생성된 **앱 비밀번호**를 사용해야 이메일 전송이 가능합니다.
*   동일 시각에 여러 작업이 몰릴 경우, 순차적으로 실행되며 약간의 시간 오차가 발생할 수 있습니다.
